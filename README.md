A simple TO-DO-List
A simple and user-friendly To-Do List application built with Python. This project helps users organize their daily tasks by allowing them to add, view, complete, and delete tasks.

## 📌 Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Save tasks (if file storage is implemented)
- 📂 Load saved tasks automatically (optional)
- ⚠️ Input validation
- 🖥️ Simple command-line interface
